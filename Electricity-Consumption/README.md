# Electricity Consumption Analytics Dashboard

## Project Overview
This project analyzes electricity consumption data for a multi-floor building to understand energy usage patterns and identify peak consumption areas.

## Tools Used
Power BI  
Excel  
Data Analysis  

## Key Insights
• Total electricity consumption  
• Floor-wise AC energy usage  
• Equipment-wise power consumption  
• Daily and weekly energy trends  

## Equipment Analyzed
RYB  
Heat Pump  
AC  
SPA  
Lift
